# Web3 Wallet Balance Checker

This is a simple Web3 project.  
It helps you check the balance of a wallet address on a blockchain network.

This project is made for beginners who want to learn how wallet balance and blockchain data work.

## What This Project Does
- Read wallet address
- Fetch wallet balance
- Show balance in console
- Save balance history to file

## Tools Used
- Node.js
- Simple JavaScript
- Public blockchain API (mock)
- Local JSON file for storage

## How To Install
1. Install Node.js
2. Download this repository
3. Open terminal in project folder
4. Run:
   npm install
5. Start the app:
   node app.js

## Folder Structure
- app.js = main file
- config.js = project settings
- balance.js = balance logic
- api.js = API calls
- saver.js = file saving
- data/ = saved balance data

## Example Use
- Set wallet address
- App shows wallet balance
- App saves balance history
